# 엔픽던 몬스터 DB

계층 - 5층 단위로 묶은 단위 (1계층 - 1~5층, 2계층 6~10층)

출현 테마 - 중간에 플레이어의 선택에 의해 엔딩 루트를 변화하는 것이 가능, 그 때 인게임 설정 상으로는 다른 구역으로 넘어가서 해당 구역의 보스를 처치하는 설정으로 넘어가기에 그것을 테마로 구분한다.
테마 기호+층 수를 통해 어떤 테마의 몇 층에서 나오는지를 확인 가능. 가령 A14일 경우 A기호를 배정받은 테마의 14층인 곳에서 등장, B27 인 경우 B 기호의 테마 27층에서 등장
테마가 바뀐다고 해서 1층으로 돌아가지 않고, 층수는 누적으로 올라간다(A테마 20층에서 보스를 잡고 조건을 만족하여 B테마로 목적지를 바꾼 경우, 그 다음 층은 B1층이 아니라 B21층이다)
’4번 협곡’ - 최초 입장 테마(A로 표기)
’무릉’ - A15에서의 보스 클리어 후 조건을 충족하여 입장 가능한 테마(B로 표기)

‘차지’와 ‘집중’ - 둘 다 몬스터가 실제 공격을 하기 전의 시전 시간이나 차지 의 경우 일부 오퍼레이터 연계기 트리거에 걸리고, 집중은 걸리지 않는다. 차지와 집중 중에는 몬스터는 일반 공격을 행하지 못하고 별도로 플레이어가 해당 행위를 중지시키지 않는 이상 해당 패턴을 끝까지 진행한다.

(특이사항 : 일반적인 경우, 플레이어의 체력 70% 이상 날릴 수 있을 정도로 강력한 피해로 설정할 예정)
사용 조건 : 페이즈 2에 돌입할 때, 한 번 쓴다.

화염 방사 : 집중 1턴, 집중 시작 시의 플레이어 방향을 향하여 2턴 간 3줄 크기로 화염을 방사하여 3x5 크기에 열기 피해(미정)를 준다. (쿨타임 5턴)
사용 조건 : 플레이어와의 거리가 3칸 이상일 때, 50% 확률로 사용한다. |
