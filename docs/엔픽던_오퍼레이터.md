# 엔픽던 오퍼레이터 로스터

> 총 25명 / 구현 파일 위치: `operators/team/`

> **오퍼레이터 시스템**: 메인 오퍼레이터 풀과 팀 오퍼레이터 풀이 따로 존재하지 않는다.
> 하나의 오퍼레이터 풀에서 플레이어가 선택한 오퍼레이터가 메인 오퍼레이터가 되고,
> 나머지 편성된 오퍼레이터들이 팀 오퍼레이터로 동작한다.
> 궁극기는 메인 오퍼레이터만 사용 가능. 패시브 중 `onBecomeMain()` 의존 효과는 메인일 때만 활성화.

---

## 직군 개요

| 직군 | 주 역할 |
|------|--------|
| **스트라이커** | 메인 딜러. 까다로운 조건의 강력한 연계기 |
| **가드** | 딜링 + 물리계. 방어불능 스택 축적 특화 |
| **캐스터** | 딜링 + 아츠계. 아츠 부착 부여 특화 |
| **서포터** | 버퍼/디버퍼. 아군 강화 + 적 약화, 회복 제한적 |
| **디펜더** | 탱커. 아군 보호/패링, 연계기로 보호/밀어내기 |
| **뱅가드** | 템포 가속. 짧은 배틀스킬 쿨타임, 연계기로 궁극기 충전 |

## 무기 유형

| 무기 | 특성 | 패널티 |
|------|------|--------|
| **한손검** | 균형잡힌 최소~최대 데미지 | 없음 (기준값) |
| **양손검** | 최소~최대 격차 최대 | 공격 시 2턴 소모 |
| **장병기** | 공격 사거리 +1칸 | 근접(바로 옆) 공격 시 데미지 감소 |
| **권총** | 사거리 최대(~6칸), 3~4칸 베스트 | 거리 멀수록 데미지↓, 근접 시 명중률↓ |
| **아츠유닛** | 항상 명중, 배틀스킬 시 충전 +1(최대 3) | 자체 데미지 낮음 |

## 아츠유닛 충전 효과

| 오퍼레이터 | 충전 효과 |
|-----------|----------|
| 자이히 | 충전 전량 소모 → 소모량 비례 배틀스킬 쿨타임 감소 |
| 질베르타 | 충전량 비례 자연 피해 (감속 적 추가 피해) |

| 펠리카 | 충전량 비례 순간이동 (충전 1당 1칸, 최대 3칸) |
| 안탈 | 충전량 비례 시간동안 물리 피해 감소 |

- **배틀스킬**: 중력 특이점 소환 (범위 내 지속 자연 피해 + 감속, 종료 시 폭발 → 자연 피해 + 자연 부착)
- **연계기** 조건: 아츠 상태이상 상태 시
- **연계기** 효과: 자연 피해 + 띄우기
- **궁극기**: 중력 혼란 구역 생성 (자연 피해 + 자연 부착 + 감속 + 아츠 취약 + 방어불능 스택 비례 아츠 취약 수치 증가)
- **충전 효과**: 충전량 비례 자연 피해 (감속 적 추가 피해)

---

### 아델리아 | 서포터 | 아츠유닛 | 구현: ✅ 완료

- **배틀스킬**: 자연 피해. 부식 상태 적 → 소모 → 물리 취약 + 아츠 취약 동시 부여
- **연계기** 조건: 강력한 일격 적중 시
- **연계기** 효과: 자연 피해 2회 + 부식 부여
- **궁극기**: 주변 적에게 10회 자연 피해
- **충전 효과**: 일정 턴 소비 → 충전량 비례 생명력 + 허기 회복

---

### 플루라이트 | 캐스터 | 권총 | 구현: ✅ 완료

- **배틀스킬**: 자연 피해 + 자연 부착 + 감속
- **연계기** 조건: 냉기 or 자연 부착 2스택 이상 시
- **연계기** 효과: 자연 피해 + 재부착 (현재 스택 +1 → 아츠 폭발)
- **궁극기**: 4회 자연 피해 (감속 시 피해 증가). 냉기/자연 2스택 이상 시 재부착

## 전기 속성 (4명)

### 아비웨나 | 스트라이커 | 장병기 | 구현: ✅ 완료

- **배틀스킬**: 부착된 모든 썬더랜스 + 강력한 썬더랜스를 회수 → 종류·수량 비례 전기 피해 (강력한 썬더랜스가 더 높은 피해)
- **연계기** 조건: 해당 적에게 최초 공격 시 (HitCounter 첫 타) or 전기 부착/감전 상태 적에게 강력한 일격 적중 시
- **연계기** 효과: 전기 피해 + 썬더랜스 3개 부착 (일정 턴 후 자동 폭발 or 배틀스킬 회수 시 피해)
- **궁극기**: 전기 피해 + 강력한 썬더랜스 1개 부착 (이후 자동 폭발 or 배틀스킬 회수 시 대량 피해)
- **특수 — 썬더랜스**: 적에게 부착형 버프. 일정 턴 후 자동 폭발 → 전기 피해. 배틀스킬 회수 시 즉시 피해 (강력한 썬더랜스는 더 높은 피해)

---

### 아크라이트 | 뱅가드 | 한손검 | 구현: ✅ 완료

- **배틀스킬**: 2회 물리 피해. 감전 상태 적 → 감전 소모 → 전기 피해 + 궁극기 충전
- **연계기** 조건: 감전 상태 or 감전 소모 시
- **연계기** 효과: 물리 피해 + 궁극기 충전 (최대 2회 분 사전 충전 가능 — 쿨타임 만료마다 1스택 적립, 최대 2스택)
- **궁극기**: 2회 전기 피해 + 전기 부착. 전기 부착 보유 적 → 소모 → 감전

---

### 펠리카 | 캐스터 | 아츠유닛 | 구현: ✅ 완료

- **배틀스킬**: 전기 피해 + 전기 부착
- **연계기** 조건: 강력한 일격 적중 시
- **연계기** 효과: 전기 피해 + 감전
- **궁극기**: 대량 전기 피해 (충전 요구량 낮음, 자주 사용 가능)
- **충전 효과**: 충전량 비례 순간이동 (충전 1당 1칸, 최대 3칸)
- ※ 첫 번째 던전 시작 시 팀 오퍼레이터 기본 보유 예정

---

### 안탈 | 서포터 | 아츠유닛 | 구현: ✅ 완료

- **배틀스킬**: 전기 피해 + 전기 취약(ElectricVulnerable) + 열기 취약(HeatVulnerable) 동시 부여
- **연계기** 조건: 대상에게 물리 이상 or 아츠 부착 부여 시 (DefenselessStack.apply / ArtsAttachment.apply → checkChainTriggers)
- **연계기** 효과: 전기 피해 + 가장 최근 연계 트리거 반복 (물리 이상이면 동일 이상 재부여, 아츠 부착이면 동일 속성 재부착)
- **궁극기**: 전기 증폭 + 열기 증폭 + 즉흥적인 천재성 획득
  - 전기/열기 증폭: 팀 연계기 포함 플레이어의 모든 전기·열기 피해에 n% 추가 적용
  - 즉흥적인 천재성: 증폭 지속 중 메인의 기본 공격 or 배틀스킬 적중 시 소량 회복 (팀 연계기로는 미발동)
- **충전 효과**: 충전량 비례 시간동안 물리 피해 감소

## 미확정/미결정 사항

- 넘어뜨리기 4스택 특수효과 (현재 넉백 1칸 예정)
- 이본 궁극기 공속 수치 (0.6턴 or 0.8턴)
- 스노우샤인 쉴드 회복 전환 비율
- 각 오퍼레이터 연계기/배틀스킬 쿨타임 수치
- 각 오퍼레이터 궁극기 최대 충전량 수치
- 첫 던전 시작 시 기본 보유 오퍼레이터 목록 (관리자 메인 + 펠리카 팀 유력)
