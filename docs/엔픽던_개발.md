## 2. 파티 시스템

- 파티 최대 **4명** (메인 1 + 팀 최대 3)
- **시작 구성**: 메인 1명 + 팀 1명 선택 후 출발
- **메인 오퍼레이터**: 플레이어가 직접 조작, 기본공격/배틀스킬/궁극기 사용 가능
- **팀 오퍼레이터**: 조작 불가, 연계기만 사용
- 던전 중 **메인 오퍼레이터 교체 없음**

## 3. 스킬 체계

### 기본 공격 & 강력한 일격

| 무기 | 콤보 단계 | 총 소요 턴 |
| --- | --- | --- |
| 양손검 | 2단계 | 4턴 (2턴×2회) |
| 그 외 | 3단계 | 3턴 (1턴×3회) |

마지막 단계 공격 = **강력한 일격** (여러 연계기 조건의 트리거)

### 배틀 스킬

- 쿨타임 방식 (매 턴 -1 카운트다운)
- 메인 오퍼레이터만 사용 가능

### 궁극기

- **충전 방식** (공격/처치/배틀스킬/연계기 발동 등 오퍼레이터별 조건)
- 메인 오퍼레이터만 사용 가능

### 연계기

- **턴 소모 없음**
- **쿨타임 있음** (매 턴 -1 카운트다운)
- 동시 활성화 시 **파티 순서대로 큐** 처리
- UI에는 항상 맨 앞 오퍼레이터만 표시

## 4. 영입 시스템

- **5층마다 보스** 처치 → **헤드헌팅 허가증** 드랍 (파티 만원이어도 드랍)
- 허가증 사용 시: 미보유 오퍼레이터 **4명 랜덤 제시**
    - 1명 선택
    - **크레디트** 소모로 리롤 (허가증당 1회)
    - 포기
- 파티 만원 시 영입 → 기존 파티원과 **교체 가능** (교체된 오퍼레이터 **영구 삭제**)
- **5n+1층 확정 상점**: 허가증 판매 (리롤 비용보다 비쌈)
- **크레디트**: 몬스터 드랍, 상자 등 (기존 픽던 골드와 동일)

## 7. 디버프 시스템

### 7-1. 스택관련형

**아츠 부착** (열기/냉기/자연/전기)

| 상황 | 효과 |
| --- | --- |
| 첫 부착 | 해당 속성 1스택 (최대 4스택) |
| 동일 속성 재부착 | **아츠 폭발** → 해당 속성 피해 + 스택 +1 |
| 다른 속성 부착 | **아츠 반응** 발생 (상태이상형 참고) |

**방어불능 스택** (물리 이상 4종)

물리 이상 종류: `띄우기` / `넘어뜨리기` / `강타` / `갑옷파괴`

| 상황 | 띄우기/넘어뜨리기 | 강타 | 갑옷파괴 |
| --- | --- | --- | --- |
| 스택 없을 때 | 방어불능 +1스택 | 방어불능 +1스택 | 방어불능 +1스택 |
| 스택 있을 때 | +1스택 + 해당 물리 피해 | 전체 소모 + 스택 비례 강력한 물리 피해 | 전체 소모 + 스택 비례 약한 물리 피해 + 물리 피해 증가 디버프 |
- **띄우기 4스택**: 특수효과 없음 (기준값)
- **넘어뜨리기 4스택**: 넉백 1칸 (추후 확정)

### 7-2. 상태이상형 (아츠 반응)

타속성 부착 적에게 아츠 부착 시 기존 스택 전량 소모 후 반응 발생

| 부착 | 반응 | 효과 |
| --- | --- | --- |
| 열기 | **연소** | 소모 스택 비례 열기 피해 지속 |
| 냉기 | **동결** | 소모 스택 비례 이동 불가 (1스택=1턴, 2~3스택=2턴, 4스택=3턴) / 고등급 면역 |
| 자연 | **부식** | 해당 적 공격 시 소량 회복 |
| 전기 | **감전** | 지속시간 동안 받는 아츠 피해 증가 |

**특수 기믹 - 쇄빙**: 동결 상태 적에게 물리 이상 → 동결 소모 + 대량 물리 피해

### 7-3. 행동제어형

`수면` / `기절` / `슬로우` 등 일반적인 CC기

### 기타 디버프

- **취약**: 받는 피해 증가 (물리/아츠/열기/냉기/전기/자연 취약)
- **허약**: 가하는 피해 감소
- **감속**: 이동속도 50% 감소

## 12. 오퍼레이터 로스터 (25명)

---

### 물리 속성 (7명)

**관리자 (Endministrator)** | 가드 | 한손검

- **연계기** 조건: 아군 연계기 적중 시 / 효과: 물리 피해 + 오리지늄 아츠 결정 부여
- **배틀스킬**: 강타
- **궁극기**: 대량 물리 피해 (오리지늄 아츠 결정 소모 시 추가 피해)
- 오리지늄 아츠 결정: 적이 물리 이상을 받을 때 소모 → 물리 피해
- ※ 가드임에도 방어불능 스택 직접 축적 불가, 팀 운용 시 물리 이상 전반에 반응하도록 조정

**여풍** | 가드 | 장병기

- **배틀스킬**: 물리 피해 + 넘어뜨리기. 방어불능 스택 없는 적에게 적중 시 물리 취약 추가 부여
- **연계기** 조건: 물리 취약 or 갑옷파괴 상태 적에게 메인의 강력한 일격 적중 시 / 효과: 물리 피해 + 메인에게 연타 버프
- **궁극기**: 물리 피해 + 넘어뜨리기 → 물리 피해 + 넘어뜨리기. 연타 소모 시 추가 대량 피해
- 연타: 다음 배틀스킬 or 궁극기 피해 1회 증가

**포그라니치니크** | 뱅가드 | 한손검

- **배틀스킬**: 물리 피해 + 갑옷파괴 + 소모된 방어불능 스택 비례 궁극기 충전
- **연계기** 조건: 강타 or 갑옷파괴로 방어불능 스택 소모 시 / 효과: 소모 스택 비례 물리 피해 + 궁극기 충전
- **궁극기**: 물리 피해 + 철의 서약 3스택 부여 (물리 이상 or 연계기 적중 시 1스택 소모 → 물리 피해, 마지막 스택 소모 시 대량 피해)

**진천우** | 가드 | 한손검

- **배틀스킬**: 물리 피해 + 띄우기
- **연계기** 조건: 적 방어불능 스택 누적 시 / 효과: 관통 베기 → 물리 피해 + 띄우기 (관통 이동 가능)
- **궁극기**: 7회 연속 대량 물리 피해
- ※ 승급 확장: 궁극기 도중 적 처치 시 주변 적으로 연장 / 처치 비례 충전 반환

**판** | 스트라이커 | 양손검

- **배틀스킬**: 물리 피해 + 띄우기
- **연계기** 조건: 적 방어불능 4스택 시 / 효과: 물리 피해 + 강타 (고배율, 관리자 강타보다 강함)
- **궁극기**: 대량 물리 피해 + 띄우기 → 넘어뜨리기
- ※ 포그라니치니크/관리자와 파티 구성 시 스택 유지 어려움 주의

**카치크** | 디펜더 | 양손검

- **배틀스킬**: 다음 적 공격 방어 (물리 100%/아츠 70% 감소) → 성공 시 물리 피해 + 방어불능 1스택
- **연계기** 조건: 적 차지 시작 시 / 효과: 물리 피해 + 높은 수치 쉴드 부여
- **궁극기**: 3회 대량 물리 피해 + 넘어뜨리기 + 허약 부여
- ※ 차지 패턴 중단 불가 / 쉴드로 맞딜 or 회피 선택

**로시** | 가드 | 한손검

- **패시브**: 없음
- **배틀스킬**: 물리 피해 + 띄우기. 방어불능 스택 보유 적에게 → 추가 열기 피해 + 늑대의 발톱 부여
- **연계기** 조건: 방어불능 스택 + 아츠 부착 동시 보유 시 / 효과: 물리 피해 + 모든 아츠 부착 소모 → 특수 띄우기 (방어불능 2스택)
- **궁극기**: 대량 열기 피해 + 열기 부착. 늑대의 발톱 상태 시 추가 열기 피해 + 회복
- 늑대의 발톱: 매 턴 물리 피해 + 받는 물리/열기 피해 증가
- ※ 물리+열기 크로스 속성, 아츠 부착→방어불능 변환 역할

### 냉기 속성 (7명)

**이본** | 스트라이커 | 권총

- **배틀스킬**: 지정 위치에 냉기 장치 설치 (3×3 범위 매 턴 냉기 충격파, 종료 시 폭발 → 냉기 피해 + 강제 동결)
- **연계기** 조건: 냉기 부착 or 자연 부착 보유 시 / 효과: 냉기 피해 + 모든 아츠 부착 소모 → 강제 동결 + 스택 비례 냉기 피해 + 궁극기 충전
- **궁극기**: 다음 5회 공격 강화 (공속 0.6~0.8턴, 추가 냉기 피해 자동 부여, 동결 시 추가 피해, 5번째 공격 대폭 강화)

**라스트 라이트** | 스트라이커 | 양손검

- **패시브**: 강력한 일격 적중 시 대상 열기 부착 흡수 → 녹아내린 불꽃 스택 획득 (최대 4스택, 4스택 시 흡수 불가)
- **배틀스킬**: 메인에게 저온 주입 부여 (턴 소모 없음). 강력한 일격 적중 시 소모 → 추가 냉기 피해 + 냉기 부착
- **연계기** 조건: 적 냉기 부착 3스택 이상 시 / 효과: 모든 냉기 부착 소모 → 스택 비례 대량 냉기 피해 + 궁극기 충전
- **궁극기**: 3회 대량 냉기 피해 + 시전 중 모든 피해 면역
- ※ 타 오퍼레이터의 궁극기 충전 효과 적용 불가, 자체 연계기로만 충전

**스노우샤인** | 디펜더 | 양손검

- **배틀스킬**: 다음 턴까지 방어 → 성공 시 냉기 피해 + 냉기 부착
- **연계기** 조건: 적 차지 시작 or 메인 HP 30% 이하 / 효과: 쉴드 부여 → 지속 종료 시 잔여 쉴드 n% 회복
- **궁극기**: 대량 냉기 피해 + 적 주변 냉기 피해 지대 생성
- ※ 카치크보다 쉴드량 낮음, 대신 쉴드→HP 전환 회복 보유

**자이히** | 서포터 | 아츠유닛

- **배틀스킬**: 지원 결정체 소환 (강력한 일격 시 추가 냉기 피해 + 아츠 증폭, 최대 3회 / 아츠 증폭 중첩 불가, 갱신만)
- **연계기** 조건: 아군 배틀스킬 적중 시 / 효과: 냉기 피해 + 냉기 부착
